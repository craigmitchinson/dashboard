## Delegation

Delegate to `explore` for any question about locating, finding, or identifying content across the project or reference material.

Delegate to `worker` for well-specified implementation: components, data layer changes, chart wiring, styling, tests.

Delegate to `test` after any worker task that touches build-affecting code.

Delegate to `reviewer` before considering a feature done, particularly for anything touching the exception classification or cost data model.

Handle directly: dashboard architecture and layout decisions, ambiguous requirements, ad hoc RPA metric definitions.
